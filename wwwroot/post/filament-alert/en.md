# Filament Alert Prototype

![Filament Alert Prototype](https://commons.wikimedia.org/wiki/Special:FilePath/SDO_SDO_20120831_171.304.blend.jpg)

The prototype uses a single rule: the alert should explain the next action before it explains the phenomenon.

In the demo, I used Solstice's code and callout styling to check whether technical notes still read like product writing. The answer depends on restraint: one threshold, one chart, one decision.

A warning that cannot be acted on is just weather in a louder voice.
