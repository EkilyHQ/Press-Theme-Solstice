# Classroom Display Pilot

![Classroom Display Pilot](https://commons.wikimedia.org/wiki/Special:FilePath/Classroom_with_projector.jpg)

The pilot succeeded when students stopped asking what the screen was and started asking whether today's image was live. That is the interface crossing from novelty into room behavior.

Teachers used quiet mode during arrivals and tracking mode during lab blocks. Nobody used alert mode, which is exactly what should happen during a calm week.

Next pilot change: add a small label explaining image delay without turning the wall into a troubleshooting panel.
