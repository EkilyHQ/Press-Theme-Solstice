# Quiet Disk Mode

![Quiet Disk Mode](https://commons.wikimedia.org/wiki/Special:FilePath/Sun_disk.jpg)

Quiet disk mode is for rooms where nobody is actively debugging. It keeps the star visible and moves every number to the edge.

The mode came from a school installation where the old dashboard looked impressive and taught almost nothing. Students walked past a wall of numbers. Teachers wanted a screen that could start a question without demanding a lecture.

This article tests Solstice's ability to keep a circular image crisp against a minimal surface. The theme's radius, shadow, and media variables make the image feel placed rather than pasted.

The best status screen is the one people can understand while walking past it.
