# Archive Mode Search

![Archive Mode Search](https://commons.wikimedia.org/wiki/Special:FilePath/Solar_archive_images.jpg)

The archive used to search by timestamp and channel. Teachers asked for loops, spots, storm, quiet day, and something that looks like fire.

We added curated question tags without hiding the technical metadata. Search can start human and become precise as the lesson develops.

Archive mode is where the interface stops being a display and becomes a teaching instrument.
